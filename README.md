# 🌸 Bloom Finance

> **Uma plataforma inteligente para organização financeira com Inteligência Artificial.**

O **Bloom Finance** foi desenvolvido para transformar o controle financeiro em uma experiência simples, intuitiva e moderna. O projeto combina **UX/UI**, **Front-End** e **IA** para permitir o registro de despesas por voz ou texto, organização automática por categorias e geração de insights sobre hábitos financeiros.

---

## ✨ Destaques

* 🤖 Assistente financeiro com IA
* 🎙️ Registro de gastos por voz
* 📊 Organização inteligente por categorias
* 📈 Insights automáticos sobre despesas
* 📁 Exportação para CSV
* 💰 Controle de valores a receber
* 📱 Interface responsiva
* ✨ Animações e microinterações inspiradas em produtos modernos

---

## 🛠️ Tecnologias

* Angular / TypeScript
* HTML5 & CSS3
* Vite
* Web Speech API
* Inteligência Artificial

## 🚀 Executando o projeto

### Clone o repositório

```bash
git clone https://github.com/sofiamasha/bloomce.git
```

### Acesse a pasta

```bash
cd bloomce
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:5173
```
